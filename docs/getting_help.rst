Getting Help
========================================================================================

The LightCurveLynx team wants to make the software as useful as possible to the entire astronomy community. We are happy to answer questions, take suggestions for improvements or features, and help you get up and running with the software.

There are a bunch of ways to get help:

- **GitHub Issues**: If you are running into errors, find a bug, need a feature, or have a question about how to use the software, you can open an issue on the [GitHub repository](https://github.com/lightcurvelynx/lightcurvelynx/issues).

- **LSST-DA Slack** Channel: If you are a member of the LSST Discovery Alliance Slack workspace, you can ask questions in the `#lincc-frameworks-lightcurvelynx <https://discovery-alliance.slack.com/archives/C08HSEG1D0D>`_ channel.

- **Office Hours**: The LINCC Framework team holds regular office hours where you can drop in to ask questions and get help. See the [LINCC Frameworks team page](https://lsstdiscoveryalliance.org/lsst-discovery-alliance-programs/lincc-frameworks/) for details.

- **FAQ**: Check out the :doc:`FAQ page <faq>` for answers to common questions.
